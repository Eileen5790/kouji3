# 起飞助手与口己1 复用分析报告

## 1. 项目概述

### 起飞助手
- **技术栈**：纯前端实现，使用原生 JavaScript 和 HTML/CSS
- **数据存储**：本地 localStorage
- **主要功能**：记录管理、数据分析、媒介和性癖管理、数据导入导出、外观管理、多语言支持

### 口己1
- **技术栈**：React Native + Expo 框架，支持 Web 和移动平台
- **数据存储**：本地 AsyncStorage
- **主要功能**：记录管理、数据分析、数据管理、设置管理、多语言支持、主题切换
