# 口己 (Kouji) - 个人日常记录应用

!\[口己 logo]\(book.png null)

## 项目简介

口己是一款开源的日记，他的特点是可以直接通过标签的选择和复用直接构建当天的日记，从而避免繁琐的写日记过程。很多人想记录每天都干了什么，记录一下每天发生了什么，但是一提笔或者打开日记软件就不知道写什么，所以本软件应运而生。（实际上我就是这样的人。）



## 主要功能

- 📝 **日常记录**：记录活动、地点、饮食和心情
- 📊 **数据分析**：可视化展示活动分布、地点分布、饮食分布和心情趋势
- 📅 **日历视图**：通过日历直观查看每天的记录和心情
- 🌙 **主题切换**：支持浅色和深色模式
- 🌍 **多语言支持**：支持中文和英文
- 💾 **数据管理**：支持数据导入/导出（CSV、JSON格式）
- 📱 **跨平台**：支持Web和Android平台

> **声明**：以上功能描述完全由AI生成，仅用于娱乐作用。实际功能可能会因版本更新而有所不同。

## 技术栈

- **前端**：HTML, CSS, JavaScript
- **跨平台框架**：Capacitor
- **数据存储**：localStorage
- **图表库**：原生Canvas API

## 快速开始

### Web 端运行

1. 克隆项目到本地
2. 进入项目目录
3. 启动本地服务器
   ```bash
   # Windows
   .\web\start-web-standalone.bat

   # 或使用PowerShell
   .\web\server.ps1
   ```
4. 在浏览器中访问 `http://localhost:8000`

### Android 端运行

1. 克隆项目到本地
2. 进入项目目录
3. 安装依赖
   ```bash
   npm install
   ```
4. 构建Android应用
   ```bash
   npx cap sync
   npx cap build android
   ```
5. 在Android Studio中打开项目并运行

## 核心功能详解

### 1. 记录功能

- **活动记录**：选择活动类型并添加详细信息
- **地点记录**：记录活动发生的地点
- **饮食记录**：记录每日饮食，支持随机推荐食物
- **心情记录**：通过表情符号记录每天的心情

### 2. 分析功能

- **活动分析**：饼图和条形图展示活动分布
- **地点分析**：饼图和条形图展示地点分布
- **饮食分析**：条形图展示饮食分布
- **心情趋势**：折线图展示心情变化趋势
- **时间筛选**：支持按周、月、年或自定义时间范围查看数据

### 3. 数据管理

- **导出数据**：支持导出为CSV和JSON格式
- **导入数据**：支持从文件或剪贴板导入数据
- **数据备份**：支持将数据复制到剪贴板进行备份

### 4. 设置功能

- **语言设置**：切换中文或英文
- **主题设置**：切换浅色或深色模式
- **日历颜色方案**：选择不同的颜色方案
- **活动管理**：添加、编辑和删除活动类型
- **地点管理**：添加、编辑和删除地点
- **饮食管理**：添加、编辑和删除食物类别和食物

## 数据安全

- 所有数据存储在本地浏览器的localStorage中，不会上传到任何服务器
- 建议定期导出数据进行备份
